# DelCaribe — Next.js 15 (App Router)

Portafolio personal listo para desplegar en **Vercel**.

## Stack
- Next.js 15 (App Router) + React 19
- Tailwind CSS
- Framer Motion

## Desarrollo local
```bash
npm install
npm run dev
```
Abre http://localhost:3000

## Deploy a Vercel
1. Sube este repo a GitHub.
2. Entra a https://vercel.com/new → importa el repo.
3. Vercel detecta Next.js automáticamente. Click **Deploy**.

No requiere variables de entorno.
